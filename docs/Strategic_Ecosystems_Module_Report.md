# Strategic Ecosystems Module Report

## GitHub Pages 发布状态说明

### 当前状态

- AI基础设施生态已经存在 GitHub Pages 公开版本。
- 半导体生态已经存在 GitHub Pages 公开版本。
- 线上 GitHub Pages 版本不是名称占位，已经包含公开摘要和详情内容。
- 本地 `develop` 分支的 `public_site/data/ecosystems.json` 比线上 Pages 当前版本更丰富。
- 例如：AI基础设施生态在本地 `develop` 的 `company_count` 是 26，线上 Pages 当前展示为 12。

### 状态定义

- 线上 Pages：已发布旧版本 / 轻量公开版本。
- 本地 `develop`：增强版本 / 待统一审查后再发布。

### 战略生态建设原则

- 当前线上 Pages 与本地 `develop` 可能存在版本差异。
- 当前战略生态建设原则是先本地完善，后统一审查发布。
- 后续建设华为生态、机器人生态、高端材料生态、船舶与国防生态、医疗科技生态时，默认只建设 Obsidian + localhost + `develop`。
- `public_site` 不自动同步内部研究内容。
- GitHub Pages 发布需要单独明确指令。
- 等七大战略生态完成后，再统一进行公开版发布审查。

### 发布边界

- 本阶段不发布 GitHub Pages。
- 本阶段不合并 `main`。
- 本阶段不修改 `public_site` 数据文件。
- 内部研究内容默认不进入公开展示层。
