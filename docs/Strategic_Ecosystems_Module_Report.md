### 与 AI基础设施生态的关系

高端材料生态为 AI服务器、数据中心、液冷系统、高速 PCB、光模块、交换机、电力系统和电子封装提供材料基础。导热材料、低介电材料、覆铜板、光通信材料和数据中心电力材料，是 AI基础设施从算力扩张走向高密度部署时的重要约束。

### 与半导体生态的关系

高端材料生态中的硅片、光刻胶、电子特气、CMP 材料、湿电子化学品、靶材、高纯石英、高纯石墨和先进封装材料，是半导体生态的底层输入。客户认证、批次稳定性、纯度、良率和国产替代率，是两个生态之间最重要的交叉跟踪变量。

### 与机器人生态的关系

机器人生态需要轻量化材料、柔性材料、传感材料、耐磨材料、稀土永磁材料、机器人线缆材料和高强度结构材料。高端材料生态决定机器人在重量、耐久、感知、执行效率和量产成本上的长期边界。

### 与船舶与国防生态的关系

船舶与国防生态依赖高温合金、钛合金、碳纤维复合材料、隐身材料、耐烧蚀材料、船舶特种钢、海洋防腐材料和特种功能涂层。高端材料生态是后续建设船舶与国防生态时需要复用的底层材料框架。

### 估值原则

- 估值是概率，不是确定性。
- 估值是区间，不是单点。
- 估值必须服务于风险控制。
- 估值必须依赖产业研究和公司研究。
- 估值必须显性化核心假设，并通过情景分析和风险折价表达不确定性。
- 估值结论必须可复查、可更新、可被事实推翻。

### 估值对象分类

- 二级市场上市公司：A股、港股、港股通和美股公司。

- 平台型资产：软件平台、AI平台、交易平台、数据平台和生态型公司。

### 文件定位

`Rachel风险控制原则` 是 Rachel Capital OS 的方法论层基石文件，用于识别、评估、控制和复盘风险。它强调风险控制不是避免波动，而是避免永久性损失、认知失真和系统性失控。

### 风险定义

Rachel Capital OS 中的风险包括永久性资本损失、核心假设被证伪、估值严重透支、产业趋势判断错误、公司竞争力恶化、财务和现金流失控、治理和关键人物风险、流动性不足、仓位过度集中、认知偏差、一级市场退出困难、组合相关性过高以及宏观和政策环境突变。

### 风险分类体系

风险分类覆盖产业风险、公司风险、估值风险、财务风险、政策与监管风险、技术路线风险、关键人物与治理风险、流动性风险、仓位风险和认知偏差风险。

### 风险等级体系

风险等级分为低、中、高、极高和待确认。每个等级对应不同处理方式，包括正常跟踪、降低假设、限制仓位、进入观察、暂缓和补充尽调。

### 风险补偿判断

风险补偿判断关注估值折价、下行空间、上行空间、风险可跟踪性、催化剂、安全边际、时间周期、退出机制和组合稳定性。

### 安全边际原则

安全边际来自估值折价、资产质量、现金流、龙头地位、技术壁垒、客户黏性、订单能见度、资产负债表安全、下行风险有限和退出路径明确。

### 仓位控制原则

仓位不是信心表达，而是风险预算表达。方法论以观察仓、标准仓、高置信仓和禁止高权重四类原则表达仓位约束，不给出具体比例。

### 一级市场风险控制

一级市场风险控制重点关注估值过高、融资依赖、商业化不确定、创始人风险、技术量产路径、客户验证、财务透明度、退出路径、后续融资稀释、股权结构和信息不对称。

### 二级市场风险控制

### 栏目调整

- 新栏目名称：合作机会。
- 旧 `company-observations` / `companies` 入口仅保留兼容跳转或路由兼容。
- 新栏目公开路径采用 `cooperation-opportunities`。

### 合作机会栏目定位

合作机会栏目用于以文章形式展示基于长期产业研究识别出的公开或脱敏产业合作方向，包括一级市场项目方向、上市公司产业需求、算力与AI基础设施合作、政府园区产业机会、技术供应链协同、出海合作和资本合作。

该栏目定位为产业研究、合作交流与商业机会对接入口，不是投资推荐栏目，不承诺融资、不进行项目背书，也不展示未授权项目敏感信息。

### 合作机会文章化发布方式

- 每一个合作机会对应一篇文章。
- 页面为文章索引页，不设置复杂子分类入口。
- 文章详情由 `public_site/data/cooperation_opportunities.json` 渲染。
- 新文章可参考 `public_site/content/cooperation-opportunities/_template.md`。

### 不再设置复杂子分类的原因

- 当前公开站点需要保持轻量、清晰、可维护。
- 合作方向会跨越多个战略生态，过早设置多层分类容易造成重复和维护成本。
- 文章化结构更适合逐篇审查公开边界，避免内部项目、公司数据库、估值或日报输入被误同步。

### 合作机会文章字段

- `slug`：文章路径标识。
- `title`：文章标题。
- `date`：发布日期。
- `ecosystem`：所属战略生态。
- `opportunity_type`：机会类型。
- `summary`：一句话摘要。
- `public_status`：公开方向 / 脱敏展示 / 待授权 / 内部储备。
- `content`：正文。
- `contact_note`：联系说明。
- `disclaimer`：免责声明。

### 公开展示边界

- `public_site` 只展示可公开或脱敏内容。
- 不公开内部公司数据库。
- 不公开估值引擎内容。
- 不公开投资决策引擎内容。
- 不公开日报事件输入内容。
- 不公开持仓、投委会内容或私密项目资料。
- 不公开未授权一级市场项目真实名称、融资金额、商业机密或未公开财务数据。
- 页面文案必须保持专业、克制，避免被理解为交易、融资或项目背书承诺。

### 与战略生态的关系

合作机会文章应关联一个或多个战略生态。战略生态用于提供产业链定位、关键环节、长期变量、潜在合作对象和公开研究背景。

### 与一级市场项目的关系

一级市场项目方向可以在合作机会栏目中以脱敏方式展示，但不得公开未授权项目真实名称、融资金额、客户清单、BP内容、财务数据或商业条款。

### 与上市公司产业需求的关系

上市公司产业需求可以在合作机会栏目中以方向、场景和需求类型表达。未经授权不得披露具体公司名称、内部需求清单、采购条件、合同信息或未公开战略计划。
