# AI基础设施生态

## 1. 生态定义

AI基础设施生态，是支撑大模型训练、推理、部署和商业化落地的底层能力体系。

它不是单一行业，而是由算力芯片、先进封装、存储、服务器、交换机、光模块、数据中心、电力、液冷、云平台和AI应用共同组成的复合生态。

在 AI 时代，基础设施的瓶颈往往决定应用扩散速度，也决定资本市场最先定价的产业环节。

## 2. 产业链结构

```text
电力
↓
数据中心 / IDC
↓
液冷 / 配电 / UPS
↓
AI服务器
↓
GPU / ASIC / HBM / 先进封装
↓
交换机 / 网络设备
↓
光模块 / 光芯片 / PCB
↓
云平台
↓
模型训练 / 推理 / AI应用
```

## 3. 核心价值链

| 环节 | 关键变量 | 观察重点 |
| --- | --- | --- |
| 算力芯片 | GPU、ASIC、国产AI芯片 | 供需缺口、价格、交付周期 |
| 存储 | HBM、DDR、SSD | HBM产能、AI服务器配套 |
| 网络 | 交换机、光模块、光芯片 | 800G/1.6T升级节奏 |
| 服务器 | AI服务器、整机制造 | 大客户资本开支 |
| 数据中心 | IDC、电力、液冷 | PUE、电力资源、建设周期 |
| 云平台 | 公有云、私有云、算力租赁 | 推理需求与商业模式 |
| 应用 | Agent、多模态、企业AI | 推理算力消耗增长 |

## 7. 与其他生态的关系

- 与半导体生态高度重叠：GPU、HBM、先进封装、制造设备。
- 与高端材料生态相关：电子化学品、封装材料、液冷材料。
- 与华为生态相关：昇腾、鲲鹏、华为云。
- 与机器人生态相关：具身智能模型和边缘推理算力。

## 8. 公开展示摘要

AI基础设施生态是 AI 时代最重要的底层能力体系之一。它决定模型训练、推理部署和应用扩散的速度，也决定算力芯片、存储、网络、数据中心、电力和云平台等产业链环节的景气度变化。
