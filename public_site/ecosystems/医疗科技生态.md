# 医疗科技生态

## 1. 生态定义

医疗科技生态是医药创新、医疗设备、AI制药、生物技术和医疗服务共同组成的长期产业方向。

它具有需求刚性、研发周期长、政策影响强和技术迭代快的特征，是资本市场中长期需要持续跟踪的重要生态之一。

## 2. 产业链结构

```text
基础研究
↓
药物发现 / 器械研发
↓
临床试验 / 注册审批
↓
生产制造
↓
商业化推广
↓
医院 / 患者 / 支付体系
```

## 3. 核心价值链

| 方向 | 关键变量 | 观察重点 |
| --- | --- | --- |
| 创新药 | 管线、临床、适应症 | 研发成功率与商业化 |
| 医疗器械 | 高端影像、手术机器人、耗材 | 国产替代与全球化 |
| AI制药 | 算法、数据、实验闭环 | 研发效率提升 |
| CXO | 订单、产能、客户结构 | 全球研发外包周期 |
| 生物技术 | 基因、细胞、蛋白 | 技术平台化 |
| 医疗服务 | 医院、支付、患者 | 政策与需求 |

## 6. 关键问题

1. 中国创新药能否形成全球竞争力？
2. 医疗器械国产替代是否进入高端环节？
3. AI制药是否能真正提升研发效率？
4. 医药板块估值修复需要哪些条件？
5. 哪些医疗科技公司具备平台型公司潜力？
6. 医疗科技如何在政策、商业化和全球化之间找到平衡？

## 7. 与其他生态的关系

- 与AI基础设施生态相关：AI制药、医疗影像AI。
- 与高端材料生态相关：生物材料、医疗耗材。
- 与机器人生态相关：手术机器人、康复机器人。
- 与半导体生态相关：医疗影像芯片、传感器。

## 8. 公开展示摘要

医疗科技生态兼具长期需求、技术创新和政策周期特征。创新药、医疗器械、AI制药和生物技术共同构成中国医疗产业升级的重要方向。
