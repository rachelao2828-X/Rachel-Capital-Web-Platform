# 半导体生态

## 1. 生态定义

半导体生态是科技产业的底层生产力系统，连接计算、通信、存储、传感、汽车、工业和AI基础设施。

它既是全球科技竞争的核心环节，也是中国科技自主可控与产业升级最重要的长期方向之一。

## 2. 产业链结构

```text
EDA / IP
↓
半导体设备
↓
半导体材料
↓
晶圆制造
↓
封装测试
↓
芯片设计
↓
终端应用
```

## 3. 核心价值链

| 环节 | 关键变量 | 观察重点 |
| --- | --- | --- |
| EDA/IP | 工具链、设计能力 | 国产替代进度 |
| 设备 | 光刻、刻蚀、薄膜、清洗、检测 | 国产设备渗透率 |
| 材料 | 光刻胶、电子气体、靶材、硅片 | 供应安全与技术壁垒 |
| 制造 | 先进制程、成熟制程 | 产能利用率与资本开支 |
| 封测 | 先进封装、Chiplet | AI芯片封装需求 |
| 设计 | AI芯片、存储、模拟、CIS、MCU | 产品周期与需求弹性 |
| 应用 | AI、汽车、消费电子、工业 | 下游景气度 |

## 7. 与其他生态的关系

- 与 AI基础设施生态直接相关：GPU、HBM、先进封装。
- 与华为生态直接相关：昇腾、鲲鹏、国产芯片链。
- 与机器人生态相关：传感器、MCU、控制芯片。
- 与高端材料生态相关：电子化学品、硅片、特种气体。

## 8. 公开展示摘要

半导体生态是全球科技产业竞争的核心底座，也是中国科技自主可控的重要方向。AI、汽车电子、先进封装与国产替代共同驱动半导体产业出现长期结构性机会。
