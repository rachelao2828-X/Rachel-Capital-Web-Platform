# 船舶与国防生态

## 1. 生态定义

船舶与国防生态是高端制造、全球贸易、海洋经济和国家安全的交汇方向。

船舶产业同时具备周期属性、制造升级属性和全球竞争格局重构属性；国防产业则具有长期需求稳定、技术壁垒高和供应链安全属性。

## 2. 产业链结构

```text
钢材 / 特种材料
↓
船舶设计
↓
动力系统
↓
船体制造
↓
船载设备
↓
军船 / 民船 / 海工装备
↓
交付与维护
```

## 3. 核心价值链

| 环节 | 关键变量 | 观察重点 |
| --- | --- | --- |
| 民船 | 集装箱船、油轮、LNG船 | 全球订单周期 |
| 军船 | 舰艇、配套系统 | 国防需求与技术壁垒 |
| 海工 | 海洋工程装备 | 油气资本开支 |
| 动力 | 船用发动机、燃机 | 国产化与技术突破 |
| 材料 | 特种钢、复材 | 高端制造能力 |
| 配套 | 雷达、通信、电子系统 | 信息化与智能化 |

- 与高端材料生态相关：特种钢、复合材料。
- 与半导体生态相关：军工电子、雷达、通信芯片。
- 与AI基础设施生态相关：智能制造、仿真设计。
- 与机器人生态相关：船厂自动化与智能制造。

## 8. 公开展示摘要

船舶与国防生态兼具周期、制造升级和全球竞争格局重构特征。中国造船产业链与国防高端制造能力的提升，是观察中国制造全球竞争力变化的重要窗口。
